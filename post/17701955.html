到达终点数字

发表于 2022-11-04 阅读次数：

题面

在x轴上给出一点target，要求从0出发，移动到target位置。移动规则如下：

  1. 每次可以选择向左或向右；
  2. 第i次移动的步数必须为i。

给出到达target的最小步数。

样例

input： 2，3，5
ans: 3，2，5

tips: -1e9 <= target <= 1e9, target != 0

思路

  1. 正负数不重要，所以取绝对值。
  2. 我们可以贪心地一直向右走；
  3. 恰好到了target，则ans = n;
  4. 越过target偶数步，则可以在1~n中选取一步x向左走【这一步向左相当于-2x，也就解释了为什么偶数和奇数要分开讨论】；
  5. 越过target奇数步，我们考虑 n+1、n+2 的奇偶性，一定要让当前位置越过target偶数步，才能使得-2x到达target。
