发表于 2022-10-19 更新于 2022-10-21 分类于 C++ 阅读次数：

编写一个简单的C++程序

类型是程序设计的最基本概念之一。一种类型不仅定义了数据元素的内容，还定义了这类数据上可以进行的运算。
程序锁处理的数据都保存在变量中，而每个变量都有自己的类型。如果一个名为val的变量的类型是T，通常我们说 “val是一个T类型变量” 或 “val具有类型T”。

编译、运行程序

初识输入输出

C++使用的iostream库中定义了两个基础类型istream和ostream，分别表示输入流和输出流。一个流就是一个字符序列，是从IO设备读出或写入IO设备的。术语“流” 的意思是：随着时间推移，字符是顺序产生或消耗的。

标准库的四个标准输入输出流：cin、cout、cerr、clog。

cout可以重定向（比如输出到文件），通过缓冲区。cerr不可以重定向，只能输出到显示器，不通过缓冲区。cerr的作用是在一些特殊的紧急情况下还可以输出（比如调用栈用完了，没有出口的递归等等）。

缓冲区的目的是减少刷屏的次数，多个字符同时输出到显示器。

endl可以刷新缓冲。在添加打印语句时，应保证一直刷新流，以防程序崩溃，输出还停留在缓冲区内。

注释简介

当修改代码时，不要忘记修改注释。
C++有两种注释：

  1. 单行注释：以双斜线 // 开始，换行符结束。
  2. 界定符对注释：以 /* 开始，*/结束。
  3. 当界定符对注释跨越多行时，最好能显示指出其内部的程序行都属于多行注释的一部分，因此建议注释内的每行都以一个星号开头。
    在 VS 中使用快捷键注释多行时采用的是 //。理解：因为要注释的代码中可能包含 */ 字符，这种情况下使用界定符对注释会产生错误，所以采用单行注释方式

控制流

while语句

while语句在执行过程中交替地检测条件和执行关联的语句，直到条件为假。

for语句

因为在循环条件中检测变量、在循环体中递增变量的模式使用非常频繁，所以 C++ 专门定义了第二种循环语句：for 语句，来简化这种模式。

读取数量不定的输入数据

使用while(cin >> x)来读取数量不定的输入，循环会一直执行到文件结束符或输入错误为止。
windows的文件结束符是Ctrl + Z，然后回车。

在编译时，最好修改一个错误就编译一次，或者最多修改了一小部分后重新编译。

类简介

类是 C++ 最重要的特性之一。
类定义了行为。类的作者决定了类类型对象上可以使用的所有操作。

初识成员函数

成员函数是定义为类的一部分的函数，也被称为方法。
通常以类对象的名义来调用成员函数：Vector.size()，即使用点运算符(.)。
