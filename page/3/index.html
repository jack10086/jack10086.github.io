到达终点数字

发表于 2022-11-04

题面

在x轴上给出一点target，要求从0出发，移动到target位置。移动规则如下：

  1. 每次可以选择向左或向右；
  2. 第i次移动的步数必须为i。

给出到达target的最小步数。

样例

input： 2，3，5
ans: 3，2，5

tips: -1e9 <= target <= 1e9, target != 0

思路

  1. 正负数不重要，所以取绝对值。
  2. 我们可以贪心地一直向右走；
  3. 恰好到了target，则ans = n;
  4. 越过target偶数步，则可以在1~n中选取一步x向左走【这一步向左相当于-2x，也就解释了为什么偶数和奇数要分开讨论】；
  5. 越过target奇数步，我们考虑 n+1、n+2 的奇偶性，一定要让当前位置越过target偶数步，才能使得-2x到达target。

KMP模式匹配算法

发表于 2022-11-03 更新于 2022-11-04 分类于 算法

KMP算法解决的是主串（s)中模式串（t）的定位问题。
不同于暴力的O(M*N)算法，KMP算法用O(M)的时间处理模式串，用O(N)的时间完成匹配。

定义

Next[i]:    “t中以i结尾的非前缀子串 与 “t的前缀” 能匹配的最长长度。
            即Next[i]=max{j}，其中j<i，且t[i-j+1 ~ i] = t[1 ~ i];当不存在这样的j时，next[i]=0。
f[i]:       “s中以i结尾的子串” 和 “t的前缀”能匹配的最大长度。

模板

应用

求主串中出现模式串的最大个数，要求两个模式串无重叠部分。

思路

正常的KMP算法模式串匹配是可以重叠的，为此我们用动态规划去重。
定义一个val数组，设s[i-m+1 ~ i] == t时，val[i]=1；否则为0。
设dp[i]表示在第i为最后一次出现时的不重叠模式串最大个数。
则

if(i>=m, val[i]) dp(i)=dp[i-m]+1
else if(i< m, val[i]) dp(i)=1
else dp(i)=0

C++ primer-2. 基本类型

发表于 2022-10-26 分类于 C++

C++定义了几种基本的内置类型，如字符、整形、浮点数等。

基本内置类型

基本内置类型包括算数类型和空类型(void)，算数类型包括字符、整数型、浮点数和布尔值。

算数类型

算数类型分为整数型和浮点型两类。字符和布尔值都属于整型。
C++定义了各类型的最小尺寸：bool未定义，char 8位，short 16位，int 16位，long 32位，long long 64位；
float是32位4字节，包括6位有效数字，double是64位8字节，包括10位有效数字。

int、short、long都是带符号类型，char由编译器决定是否有符号。
C++提供了几种字符类型：

  1. char：一个char的空间应确保可以存放机器基本字符集中任意字符对应的数字值，即一个char的大小和一个机器字节一样。
  2. wchar_t：宽字节，用于拓展字符集，wchar_t确保可以存放机器最大扩展字符集中的任意一个字符。
  3. char16_t和char32_t：为Unicode字符集服务。

类型选择：

  • 明确数值不可能为负时，选用无符号类型。
  • 整数运算用int，数值太大用long long，不用short和long
  • 浮点数运算用double。float和double的计算代价相差无几。

例题

和至少为 K 的最短子数组

题面

给你一个整数数组nums和一个整数k，找出nums中和至少为 k 的最短非空子数组。
并返回该子数组的长度。如果不存在这样的子数组，返回-1 。

子数组 是数组中 连续 的一部分。

提示

1 <= nums.length() <= 1e5
-1e5 <= nums[i] <= 1e5
1 <= k <= 1e9

注意

该数组元素可能为负，故而其区间和不具有“单调性”，因此无法使用二分查找、滑动窗口来解决问题。
例如：

nums = [44,-25,75,-50,-38,-42,-32,-6,-40,-47]
k = 19

二分时，mid=5并不能使得s[i]-s[i-mid]>=19，但mid=1却可以。

编写一个简单的C++程序

类型是程序设计的最基本概念之一。一种类型不仅定义了数据元素的内容，还定义了这类数据上可以进行的运算。
程序锁处理的数据都保存在变量中，而每个变量都有自己的类型。如果一个名为val的变量的类型是T，通常我们说 “val是一个T类型变量” 或 “val具有类型T”。

编译、运行程序

初识输入输出

C++使用的iostream库中定义了两个基础类型istream和ostream，分别表示输入流和输出流。一个流就是一个字符序列，是从IO设备读出或写入IO设备的。术语“流” 的意思是：随着时间推移，字符是顺序产生或消耗的。

标准库的四个标准输入输出流：cin、cout、cerr、clog。

cout可以重定向（比如输出到文件），通过缓冲区。cerr不可以重定向，只能输出到显示器，不通过缓冲区。cerr的作用是在一些特殊的紧急情况下还可以输出（比如调用栈用完了，没有出口的递归等等）。

缓冲区的目的是减少刷屏的次数，多个字符同时输出到显示器。

endl可以刷新缓冲。在添加打印语句时，应保证一直刷新流，以防程序崩溃，输出还停留在缓冲区内。

1 内存分区模型

C++程序在执行时，将内存大方向划分为4个区域

  • 代码区：存放函数体的二进制代码，由操作系统进行管理的
  • 全局区：存放全局变量和静态变量以及常量
  • 栈区：由编译器自动分配释放, 存放函数的参数值,局部变量等
  • 堆区：由程序员分配和释放,若程序员不释放,程序结束时由操作系统回收

内存四区意义：

不同区域存放的数据，赋予不同的生命周期, 给我们更大的灵活编程
